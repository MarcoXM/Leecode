# 1653. Minimum Deletions to Make String Balanced

```python

class Solution:
    def decrypt(self, code: List[int], k: int) -> List[int]:
        n = len(code)
        code = code + code
        ans = [0] * n
        if k == 0 :return ans
        elif k > 0:
            for i in range(n):
                ans[i] = sum(code[i+1:i+k+1])
        else:
            for i in range(n):
                ans[i] = sum(code[i+ n+k:i+n])
                
        return ans
```
