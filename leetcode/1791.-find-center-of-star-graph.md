# 1791. Find Center of Star Graph

```python

class Solution:
    def areAlmostEqual(self, s1: str, s2: str) -> bool:
        if s1 == s2:return True
        cnt  = 0
        for a, b in zip(s1,s2):
            if a != b:
                cnt += 1
                
        if sorted(s1) == sorted(s2) and cnt <= 2:
            return True
        return False
```
