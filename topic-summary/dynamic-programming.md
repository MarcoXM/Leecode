**DP套路\(III\): 双序列型**

给出两个序列s和t（数组/字符串），让你对它们搞事情。

* Longest Common Subsequences 
* Shortest Common Supersequence
* Edit distances
* ...

套路：

* 定义dp\[i\]\[j\]：表示针对s\[1:i\]和t\[1:j\]的子问题的求解。
* 千方百计将dp\[i\]\[j\]往之前的状态去转移：dp\[i-1\]\[j\], dp\[i\]\[j-1\], dp\[i-1\]\[j-1\]
* 最终的结果是dp\[m\]\[n\]\
